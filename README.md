# git-tutorial

how to use git?

git add . && git commit -m "blabla"
git push
if you want to update and push the updated codes on github

git pull
if there are some changes on your github repositories use this command to pull request and get the updated repo. note that if there are changes on the github repo you need to use the git pull command

git push --set-upstream origin feature/contacts
when you create a new branch for example a new feature you can use this command to continuosly push updates and changes on your selected branches
